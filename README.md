<h2>Instalasi</h2>

1. <h5>Clone Repository</h5>
    
```
    - git clone https://github.com/junicotandiago198/Laravel-Blog.git
    - cd Laravel-Blog
    - composer install
    - cp .env.example .env
```
    
2. <h5>Buka .env lalu ubah baris berikut sesuai dengan databasemu yang ingin dipakai</h5>
```
   - DB_PORT=3306
   - DB_DATABASE=laravel
   - DB_USERNAME=root
   - DB_PASSWORD=
```
